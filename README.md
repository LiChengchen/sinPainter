#### 可调节绘图总点数、精度（采样率）
若采样频率继续减少到一个周期内采不到两个点时，我们将无法从采样点恢复回原始信号。

实际上，根据奈奎斯特采样定理，当采样频率大于信号中最高频率2倍时，采样之后的数字信号完整地保留了原始信号中的信息。
